# doc2social

## Skills

### gzh-layout-cloner

提取公众号文章的 CSS 排版结构，保存为可复用模板；也可以将 Markdown 套入已有模板，生成可直接复制到公众号编辑器的 inline-style HTML。

内置公众号模板默认使用“绿色科技”，也支持苹方周刊、极简墨白、极简代码和莫兰迪。

### note-to-xhs

将一篇 Obsidian Markdown 笔记渲染成可发布的小红书 3:4 图文卡片，并生成小红书标题、正文文案和标签。

支持极简风、X长文、莫兰迪、彩色卡、茶色笺、胡桃木、点阵纸。短文可单卡导出，长文自动分页。

## 安装

```bash
git clone https://github.com/AlineFan/doc-to-social-render.git
cd doc-to-social-render
```

在 Codex 中打开该目录即可使用项目内的 `.agents/skills/`：

- `gzh-layout-cloner`
- `note-to-xhs`
