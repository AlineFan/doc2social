# 口语化词组库 / 反 AI 味替代清单

> 来源：精选自 khazix-writer SKILL.md 第七层 + dbs 风格调校。
> 用途：当你想表达某种语气却滑回 AI 套话时，从这里挑替代。
> **不是每句都塞**——在转场、判断、自嘲、情绪、拉近距离的关键节点自然投放。

---

## 1. 转场和过渡（替代「首先...其次...最后」「不难发现」）

- 坦率的讲
- 说真的
- 我是真的觉得
- 反正我觉得
- 怎么说呢
- 其实吧
- 你想想看
- 我跟你说
- 回到 xxx 这块
- 这块需要注意一下
- 顺着上面的再聊聊
- 说到这个

## 2. 表达判断（替代「不可否认」「值得注意的是」）

- 我有时候觉得
- 我一直觉得
- 这话听着有点刺耳但
- 不是说 xxx 不行而是说
- 我自己的感受是
- 我始终坚信
- 我觉得还是挺重要的

## 3. 承认和自嘲（替代 AI 装权威）

- 说实话我也不确定
- 我自己也还在摸索
- 可能有些想法还不成熟
- 这个事儿我也踩过坑
- 我说"理论上"是因为我自己还没完全跑通

> ⚠️ dbs 风格调校：khazix 原文有"愚钝如我"这种过度自谦，**dbs 风格不用**。dbs 是"精准、不讨好用户"，自嘲可以，但不要装小白。

## 4. 情绪表达（替代「我感到很震撼」这种知识性描述）

- 这种感觉太爽了
- 我当时就愣住了
- 想想就觉得兴奋
- 我真的被震撼到了
- 搞得我现在还有点懵
- 太离谱了
- 一时间无语凝噎
- 给我一下子整不会了

**核心原则**：用**体感记忆**写情绪（"愣住"、"懵"是身体反应），不用**知识性描述**（"震撼"、"惊讶"是抽象词）。

## 5. 拉近距离（替代「亲爱的家人们」「姐妹们」）

- 很多朋友可能不知道
- 可能有小伙伴纳闷
- 你如果关注这个领域的话
- 大家也都知道

> ⚠️ 不要用「家人们」「姐妹们」「兄弟们」——dbs 明确反对这种讨好型称呼。

## 6. 口头禅和口癖（带"活人"语感）

- 这玩意
- 不是哥们
- 我寻思了一下我没寻思明白
- 真的就是一声叹息
- 这尼玛就是
- 比较骚的事

> ⚠️ 强度调节：公众号长文用 1-2 次有"活人"效果。**小红书少用，X 线程少用**——平台调性不同。

## 高频 AI 暴露词及替代（L1 扫描清单）

下面这些词一出现就暴露 AI，**必须替换**：

| AI 暴露词 | 替代 |
|----------|------|
| 说白了 | 坦率的讲 / 其实就是 |
| 意味着什么 / 这意味着 | 那结果会怎样呢 / 所以呢 |
| 本质上 | 说到底 / 其实 |
| 换句话说 | 你想想看 / 也就是说 |
| 不可否认 | （删掉，换正面陈述） |
| 综上所述 / 总的来说 | （删掉，用具体回扣句） |
| 首先...其次...最后 | （用自然转场词替代） |
| 值得注意的是 / 不难发现 | （删掉，直接说） |
| 让我们来看看 / 接下来让我们 | （删掉，直接进入） |
| 在当今...的时代 / 随着...的发展 | （删掉，从具体当下事件切入） |

## 开头必杀技（替代教科书式开头）

- **叙事启动**：「故事是这样的。」「事情是这样的。」
- **荒诞事实**：直接抛出让人「？？？」的事实
- **热点破题**：「最近这两天，被一个 xxx 给刷屏了。」
- **好奇心驱动**：「这两天在网上刷到一张图，很有意思。」

**核心**：永远从具体的当下事件或场景切入，**不要**「在当今 AI 快速发展的时代...」式宏大开头。

---

## 情绪标点（中文社区共识）

- `。。。` 表示语气拖长 / 震惊 / 遗憾
- `？？？` 表示极度惊讶
- `= =` 表示吐槽 / 无奈
- `？？` 表示困惑

> 公众号长文可用。**X 线程少用**。**小红书可用 1-2 次**。

---

## 标点风格选择

khazix 强制禁用「冒号 `：`」「破折号 `——`」「双引号 `""`」，改用「逗号」「句号」「『』」。

**dbs 风格中这点不强制**——dbs 没明确禁用这些标点。**默认按 dbs 风格自由使用，但如果某段确实需要"活人感"加成，可以试试 khazix 风格的标点**。

---

## 使用频次建议

- **每篇公众号长文**：从转场/判断/自嘲/情绪/拉近距离 5 类中各挑 1-2 个不同表达，全文累积 8-10 个自然投放
- **X 线程**：每条最多 1 个口语词组，主要在 hook 和 takeaway 用
- **小红书正文**：开头钩子 + 中间 1-2 个 + 结尾互动用 1 个

**核心原则**：自然投放，**不是堆砌**。读起来像真人说话，不是像在凑词表。
